# Vision · 앞으로의 방향

> 이 문서는 **확정된 로드맵이 아닙니다.** 만든 사람이 *이런 방향으로 가면 좋겠다*고 생각하는 큰 그림. 커뮤니티 피드백을 받아 형태가 잡힙니다.

## 지금까지

**v0.1.0 — 정직한 로컬 트레이 앱.**
하루의 코딩을 우주로 시각화한다. 모든 데이터는 로컬. 코드 공개. 광고 없음. 텔레메트리 없음.

## 다음 단계 — v0.x 단기

- **주간 다이제스트**: 일주일 우주를 PNG로 export. 자기 SNS에 자유롭게 공유.
- **추가 Provider**: Cursor · Gemini Code · etc.
- **시간대 업적**: Night Owl, Early Bird (엔진에는 키 이미 예약돼있음).
- **macOS Developer ID + Windows EV 서명**: 첫 실행 시 OS 경고 사라짐.

## 중기 — v1.0 ~ v1.x

### 1. 커뮤니티 (Opt-in only)

전용 웹페이지에서 **선택적으로** 본인의 우주를 공유할 수 있는 공간:

- **갤러리 공유**: 마음에 드는 별자리/우주를 게시. 다른 사람의 작품 둘러보기.
- **통계 비교**: 익명화된 평균과 본인 데이터 비교 ("이번 주 토큰 사용량 상위 20%" 등).
- **개발자 커피챗**: 다른 사람의 작업 우주 보면서 가볍게 잡담하는 공간. Discord 또는 자체 채팅.

**핵심 원칙:**
- **공유는 100% opt-in.** 기본은 share 안 함. 사용자가 명시적으로 토글한 우주만 업로드.
- **익명 가능.** 닉네임 + 우주만 보임. 실명/이메일 강제 안 함.
- **토큰 원시 데이터 절대 안 보냄.** 별 개수, 행성 종 같은 *시각 결과물* 만 공유. 어떤 코드를 짰는지는 영원히 로컬.

### 2. 시즌 — 코스메틱 옵션

매 분기마다 새로운 시각 테마를 선택적으로 사용 가능:

- **봄**: 벚꽃잎 별, 부드러운 핑크 팔레트
- **할로윈**: 호박 톤 별, jack-o'-lantern 행성
- **겨울**: 눈송이 별 모양, 결정 행성
- **사이버펑크**: neon 청록/마젠타, glitch 별 effect

**핵심 원칙:**
- 본 앱은 **영원히 무료**. 시즌 팩은 **부가 cosmetic** 일 뿐.
- 시즌 팩 없는 사용자도 **모든 기능 100% 사용 가능** — 핵심 기능 잠금 안 함.
- 본인 우주에만 적용. 공유 시 보이는 모습은 (커뮤니티 페이지에선) 기본 시각으로 normalize.
- GitHub Sponsor는 **모든 시즌 팩 영구 포함**.

이 부분은 사이드 프로젝트를 더 오래 지속할 수 있게 하는 보조 수단입니다. *"이거 무료로 만들어주려고 본업 끝나고 매일 밤 코딩하는데, 시즌 팩으로 그걸 더 길게 갈 수 있다면 좋겠어"* — 이게 진심.

**아직 확정 X — 어떤 시즌이 매력적일지, 가격이 얼마면 자연스러울지, 커뮤니티 피드백 받아 정합니다.**

## 장기 — v2.0+

- **모바일 컴패니언 (read-only)**: 출퇴근 중에 어제 우주를 다시 보거나 갤러리 둘러보기.
- **팀 우주 (옵션)**: 회사/스터디 그룹에서 익명화된 집합 우주를 합쳐서 한 화면으로 — "이번 주 우리 팀의 총 우주". 개인 우주는 그대로 분리 유지.
- **API**: 본인의 토큰 데이터에 프로그래매틱 접근 (예: Obsidian / Notion에 매일 우주 자동 임베드).

## 현재 시점 원칙

지금 시점에서 명확한 것들 (모델은 시간이 지나면서 진화할 수 있습니다):

- **데이터는 로컬 우선.** 사용자의 토큰 원시 로그가 외부로 나가는 건 사용자가 명시적으로 opt-in한 경우만 — 자세히는 [Privacy](privacy.md).
- **현재 빌드의 코어 기능은 무료 + 오픈소스.** 별, 행성, 도감, 갤러리, 별자리, 자동 업데이트, i18n.
- **시즌 코스메틱은 사이드 옵션.** 코어 기능에 영향 X, opt-in. 가격/형태는 커뮤니티 피드백으로 결정.
- **투명성**: 모델 변화가 있으면 사용자에게 사전 공지 + 기존 기능 유지.

> 이 섹션은 "현재 빌드 기준의 약속"이며, 향후 비즈니스 모델 진화 시 변경될 수 있음. 변경할 때는 release notes / 인앱 공지로 분명히 알립니다.

## 어떻게 도와줄 수 있나

- **Issue/Discussion 참여**: 위 아이디어 중 마음에 드는 것 / 별로인 것 의견.
- **PR**: 추가 Provider, 새 행성 종, 시각 effect 개선, i18n.
- **Sponsor**: GitHub Sponsor가 활성화되면 후원 (v1.0.0 시점 예정).
- **그냥 쓰면서 알려주기**: "이건 잘 동작했다 / 이건 안 됐다" — 가장 가치 있음.

## 만든 사람의 솔직한 입장

이 앱은 본업 끝나고 저녁 시간에 만들고 있는 사이드 프로젝트입니다. 그렇다고 영원히 무료 봉사로만 지속 가능하다고 보지는 않습니다. 그래서 현재 시점의 방향은:

- **현재 빌드의 코어는 무료/오픈** — FSL 라이선스 + 2년 후 Apache 2.0 전환.
- **시즌 코스메틱과 opt-in 커뮤니티를 우선 탐색** — 프로젝트가 자기 비용을 일부 감당할 수 있을 만한 가벼운 모델부터.
- **사용자 신뢰가 최우선** — 모델 변경이 있으면 미리 공지하고 기존 기능은 유지합니다.

장기적으로 어떤 모델이 자연스러울지는 커뮤니티와 함께 정합니다. 의견 환영.
